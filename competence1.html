Compétence1: Programmation en HTML

Le HTML est un language de programmation utiliser pour developper des pages web.Au moment où je vous parle, je suis capable de créer des site web avec ce language de programmation. De nos jours avec l'avancée majeure du numérique, les commerciales éttendent leur connaissance au grand public grâce à une interface numérique qui est un site web.
Je suis encore en cours d'apprentissage de ce language fascinant qu'est le HTML et, bien que je ne sois pas capable de developper d'enorme inerface numérique pour le moment, je peux néanmoins developper de peti site web(exemple: le site web d'un magasin de vente électronique.)
