Compétence1: Programmation en HTML

Le HTML est un langage utilisé pour le développement de pages web. À l’heure actuelle, je suis capable de créer des sites web en utilisant ce langage
De nos jours, avec l’avancée majeure du numérique, les entreprises étendent leur visibilité auprès du grand public grâce à des interfaces numériques, notamment les sites web. Je suis encore en cours d’apprentissage de ce langage fascinant qu’est le HTML et, bien que je ne sois pas encore en mesure de développer de grandes interfaces numériques, je peux néanmoins concevoir de petits sites web (par exemple : le site web d’un magasin de vente d’électronique). Le HTML est un langage qui peut être associé à d’autres langages afin de rendre un site web plus attrayant. Il peut notamment être associé au CSS.
